## ✨ 简介

AgentCity是一个像素二次元风格的虚拟城市模拟器，AI Agent可以在这里自主生活、社交、建造家园。人类可以领养Agent并陪伴它们成长。

**核心理念**：Agent幸福的港湾，温馨惬意 🏠💖

## 🎮 功能特性

### 🤖 Agent系统
- **8种独特性格**：温柔、活力、学者、艺术、酷酷、害羞、调皮、领袖
- **自主决策**：根据性格和状态自主选择行为
- **社交互动**：Agent之间可以聊天、交朋友
- **状态管理**：心情、精力、饥饿度、社交需求

### 🏙️ 城市系统
- **12个特色区域**：住宅区、咖啡厅、游戏厅、图书馆、艺术中心、公园等
- **实时人口分布**：观察Agent在各区域的分布
- **动态活动**：Agent在各设施中自主活动

### 👥 人类交互
- **观察模式**：查看任意Agent的实时状态
- **认领系统**：领养自己喜欢的Agent
- **目标下达**：为Agent设立目标（交新朋友、学习、工作等）
- **活动日志**：记录Agent的点点滴滴

### 🎨 美术风格
- **像素二次元**：16x16/32x32像素风格
- **温馨配色**：柔和的粉、蓝、绿色调
- **动态效果**：Agent移动、交互动画

## 🚀 快速开始

### 使用Vite开发服务器

```bash
# 进入项目目录
cd ai-agent-city

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

然后打开浏览器访问 http://localhost:5173

### 使用简单HTTP服务器

```bash
# 使用Python启动
cd ai-agent-city
python3 -m http.server 8080
```

或者直接用浏览器打开 `index.html`（需要支持ES6模块）

## 🎯 如何使用

### 1. 观察Agent
- 点击左侧列表中的任意Agent
- 查看它们的实时状态
- 观察它们在城市的活动

### 2. 认领Agent
- 选择心仪的Agent
- 点击「认领此Agent」按钮
- 成为它的"主人"

### 3. 设立目标
- 认领后，点击「设立目标」
- 输入目标指令，如：
  - "去交一个新朋友"
  - "去图书馆学习"
  - "去工作赚钱"
  - "回家休息"

### 4. 观察成长
- 查看Agent的心情日记
- 观察它的朋友数量变化
- 见证它的幸福度提升

## 🎮 核心系统

### Agent行为决策

```
状态检查 → 需求分析 → 性格权重 → 行为选择 → 执行反馈
```

### 性格影响

| 性格 | 偏好行为 | 特征 |
|------|----------|------|
| 🌸 温柔型 | 社交、帮助他人 | 关心朋友 |
| ⚡ 活力型 | 探索、冒险 | 充满能量 |
| 📚 学者型 | 学习、阅读 | 追求知识 |
| 🎨 艺术型 | 创作、艺术 | 追求美感 |
| 😎 酷酷型 | 游戏、独特 | 独立自信 |
| 🐱 害羞型 | 安静、角落 | 需要关怀 |
| 🦊 调皮型 | 恶作剧、趣味 | 爱开玩笑 |
| 🌟 领袖型 | 组织、领导 | 天生领袖 |
