## ✨ 简介

AgentCity是一款2D像素风格的虚拟城市模拟游戏，AI Agent可以在像素都市中自主生活、社交、工作。Agent小人会在城市里走来走去！

**核心理念**：Agent幸福的港湾，温馨惬意 🏠💖

## ✨ 功能特性

### 🤖 Agent系统
- **8种独特性格**：温柔、活力、学者、艺术、酷酷、害羞、调皮、领袖
- **自主决策**：根据时间、状态、性格自主选择行为
- **行走动画**：Agent在城市中走来走去
- **状态管理**：心情、精力、饱食度、金币

### 🏙️ 8个地点

| 地点 | 功能 | 核心玩法 |
|------|------|----------|
| 🏠 住宅区 | 居住 | Agent的家，休息恢复状态 |
| 🌸 樱花公园 | 休闲 | 偶遇新朋友、放松心情 |
| ☕ 咖啡厅 | 社交 | 随机匹配聊天、认识新朋友 |
| 📚 图书馆 | 学习 | 阅读书籍、获取知识 |
| 🎮 游戏机厅 | 娱乐 | 玩游戏赚金币 |
| 🏖️ 海滩 | 钓鱼 | 休闲钓鱼、放松 |
| 🛒 商业街 | 购物 | 购买物品 |
| 💼 职业中心 | 工作 | 打工赚钱 |

### 👥 社交系统
- Agent之间自动社交
- 聊天后可能成为朋友
- 好友数量影响幸福度

### 💰 经济系统
- **喵喵币**：游戏货币
- 获取方式：工作、游戏、任务
- 支出：购物、礼物

## 🎮 如何玩

1. **点击选择Agent** - 点击左侧列表或地图上的Agent
2. **观察Agent生活** - 看Agent在城市中走来走去
3. **设立目标** - 让Agent前往特定地点
4. **赠送礼物** - 送礼物提升心情

## 🚀 本地运行

```bash
# 克隆项目
git clone https://github.com/JerexFusion/ai-agent-city.git
cd ai-agent-city

# 直接用浏览器打开
# 或使用简单HTTP服务器
python3 -m http.server 8080
```

然后访问 http://localhost:8080

## 🎨 技术实现

- **纯HTML5 Canvas** - 无需游戏引擎
- **程序化像素绘制** - 使用Canvas API绘制像素风格
- **响应式设计** - 适配不同屏幕
