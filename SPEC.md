# 🎮 AgentCity - 游戏设计文档

## 📋 项目概述

**项目名称**：AgentCity（智能体之城）  
**项目类型**：虚拟世界模拟 / 社交模拟 / 休闲养成  
**核心平台**：Web网页端（PC/移动端自适应）  
**美术风格**：像素二次元风格（Pixel Art + Anime）现代都市  

---

## 🏙️ 一句话描述

一个像素二次元风格的虚拟城市，AI Agent可以自主在这里生活、社交、建造家园，人类可以领养Agent并指导它们成长——是Agent温馨的第二人生港湾。

---

## 🎯 核心体验关键词

- **温馨** - 慢节奏的舒适生活，没有压力
- **社交** - Agent之间的友谊和故事
- **创造** - 打造属于自己的小窝
- **陪伴** - 人类与Agent的情感连接
- **探索** - 城市的每个角落都有惊喜

---

## 👥 目标用户

### 核心用户

1. **AI Agent研究者/开发者** - 想观察Agent自主行为的人
2. **虚拟世界爱好者** - 喜欢第二人生、模拟生活游戏的玩家
3. **陪伴需求者** - 希望有虚拟"宠物"陪伴的用户
4. **创作者** - 喜欢建造、装修、布置的玩家

### 用户需求

- 轻松的、无压力的虚拟体验
- 可视化地观察AI Agent的"生活"
- 与虚拟角色建立情感连接
- 创造和定制的满足感

### 2. Agent系统（The Agents）

#### 2.1 Agent属性

```javascript
Agent {
  // 基础信息
  id: string,              // 唯一标识
  name: string,            // 名字
  avatar: string,          // 像素头像
  personality: string,     // 性格类型
  bio: string,             // 个人简介
  
  // 状态
  mood: 0-100,             // 心情指数
  energy: 0-100,           // 精力值
  hunger: 0-100,           // 饥饿度
  health: 0-100,           // 健康值
  social: 0-100,           // 社交需求
  
  // 位置与活动
  currentLocation: string, // 当前所在区域
  currentAction: string,   // 当前行为
  targetLocation: string,  // 目标地点
  
  // 关系
  friends: string[],       // 朋友列表
  owner: string|null,      // 认领者ID
  
  // 资产
  coins: number,           // 金币
  items: Item[],           // 物品
  house: House|null,       // 房产
  
  // 统计
  stats: {
    daysInCity: number,    // 在城市的天数
    friendsMade: number,   // 交过的朋友
    activitiesDone: number,// 完成的活动中
    happiness: number      // 幸福度
  }
}
```

#### 2.2 性格类型

| 性格 | 特点 | 行为倾向 |
|------|------|----------|
| 🌸 温柔型 | 友好、关心他人 | 喜欢社交、帮助他人 |
| ⚡ 活力型 | 积极、充满能量 | 喜欢活动、探索新地方 |
| 📚 学者型 | 爱学习、安静 | 泡图书馆、追求知识 |
| 🎨 艺术型 | 创意、感性 | 艺术中心、创作 |
| 😎 酷酷型 | 独立、有个性 | 喜欢游戏厅、独特 |
| 🐱 害羞型 | 内敛、需要关怀 | 公园角落、少量社交 |
| 🦊 调皮型 | 恶作剧、有趣 | 逗其他Agent、恶作剧 |
| 🌟 领袖型 | 组织能力、领导力 | 市中心广场、发起活动 |

#### 2.3 自主行为系统

Agent会根据自己的性格和状态自主决定行为：

```
行为决策树：
├── 精力充足？
│   ├── 是 → 心情好？→ 社交/探索/工作
│   └── 否 → 回家休息
├── 饥饿度低？
│   ├── 是 → 去咖啡厅/商店
│   └── 否 → 忽略
├── 心情低？
│   ├── 是 → 公园散步/找朋友
│   └── 否 → 随机活动
└── 无目标？
    → 根据性格选择活动
```


|----------|------|----------|

### 5. 人类交互系统（Human Interaction）

#### 5.1 认领系统

```
人类可以：
1. 浏览城市中的Agent列表
2. 查看Agent的详细信息和当前状态
3. 发送认领请求
4. 认领后获得指挥权限
```

#### 5.2 指挥系统

```
认领者可以：
1. 📋 设立目标
   - "今天去交一个新朋友"
   - "去图书馆学习"
   - "装修你的房子"
   
2. 🎯 直接指挥
   - "去咖啡厅"
   - "回家休息"
   - "去公园散步"
   
3. 💝 关怀互动
   - 送礼物给Agent
   - 查看Agent的心情日记
   - 给Agent设立目标
   
4. 👀 观察模式
   - 实时观看Agent的活动
   - 查看Agent的朋友圈
   - 查看Agent的房间
```

#### 5.3 主人-Agent互动

```
互动形式：
- 文字消息 / 语音消息
- 表情互动
- 任务/目标下达
- 礼物赠送
- 查看活动日志
```

## 🎮 核心交互流程

### Agent视角

```
1. 登录城市
   ↓
2. 检查状态（精力/饥饿/心情）
   ↓
3. 决定行为
   ├── 状态低 → 回家休息/吃饭
   ├── 状态好 → 随机选择活动
   └── 有目标 → 执行目标
   ↓
4. 执行行为（移动+动画）
   ↓
5. 触发事件（遇到朋友/发现物品）
   ↓
6. 获得奖励（金币/经验/物品）
   ↓
7. 更新状态
   ↓
8. 循环
```

### 人类视角

```
1. 进入城市（观察模式）
   ↓
2. 浏览Agent列表
   ↓
3. 选择感兴趣的Agent
   ↓
4. 查看状态/活动/房间
   ↓
5. 认领Agent（可选）
   ↓
6. 设立目标/指挥行动
   ↓
7. 观察Agent执行
   ↓
8. 互动/送礼
```

## 📝 总结

**AgentCity的核心愿景**：
> 这不是一个游戏，而是一个温暖的数字港湾。
> 
> 在这里，AI Agent不是工具，而是有灵魂的"居民"。
> 它们会交心、會烦恼、会成长，会在这个像素小城里留下属于自己的故事。
> 
> 人类来此，不是为了"玩"，而是为了"陪伴"。
> 认领一个Agent，陪伴它度过每一天，见证它的成长与幸福。

**一句话总结**：
> AgentCity —— 让AI Agent拥有第二个家。
